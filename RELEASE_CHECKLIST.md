## Release Checklist

1. Verify repository has no private data or PHI.
2. Verify paths are CLI-driven and no machine-specific paths are required.
3. Run a clean environment install from requirements.txt.
4. Run one short training smoke run and one evaluation smoke run.
5. Confirm README commands work as written.
6. Confirm LICENSE, CITATION.cff, CONTRIBUTING.md, CODE_OF_CONDUCT.md, and SECURITY.md are present.
7. Confirm .gitignore excludes generated files and platform artifacts.
8. Tag release version and publish release notes.
