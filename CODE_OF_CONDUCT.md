## Code of Conduct

This project adopts a standard community code of conduct.

### Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone.

### Expected Behavior

- Be respectful and constructive.
- Accept feedback gracefully.
- Focus on evidence-based technical discussion.

### Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Publishing private information without permission.
- Any abusive or intimidating conduct.

### Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, and other contributions that are not aligned with this code of conduct.

### Scope

This code of conduct applies in all project spaces, including issue trackers, pull requests, and discussions.
